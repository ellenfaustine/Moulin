# Moulin Magazine

Moulin Magazine is a multi-page, content-driven digital magazine website showcasing editorial content across Fashion, Beauty, Culture, Lifestyle, and Health.

## Features
- Multi-page HTML architecture
- Responsive layout using Bootstrap and custom CSS
- Reusable navigation and footer components
- Dynamic content rendering using JavaScript and Handlebars templates

## Tech Stack
- HTML
- CSS
- JavaScript
- Bootstrap
- Handlebars

## How to Run
Open `index.html` in a browser  
or serve the project using a local web server (recommended).
